Política de Privacidad – Jepilogo

Fecha de última actualización: 15/11/2025

1. Introducción

Esta Política de Privacidad describe cómo recopilamos, utilizamos, almacenamos y protegemos los datos personales procesados a través de nuestra aplicación de automatización basada en Inteligencia Artificial (en adelante, “la Aplicación”). Al utilizar la Aplicación, aceptas las prácticas descritas en esta Política.

Nombre del responsable: Jean Pierre Londoño
Correo de contacto: jepilogo@hotmail.com
Finalidad del servicio: Automatizaciones de flujos de trabajo mediante IA, integraciones API y procesamiento automatizado de mensajes y datos.

3. Datos que recopilamos

La Aplicación solo recopila los datos estrictamente necesarios para ofrecer sus funcionalidades. Entre ellos:

3.1 Datos proporcionados por los usuarios

  • Nombre de usuario
  • Correo electrónico
  • Identificadores de cuenta (por ejemplo: ID de Instagram, ID de WhatsApp, ID de Facebook)
  • Información de perfil público
  • Mensajes recibidos en plataformas conectadas (cuando el usuario activa automatizaciones de mensajería)
  • Archivos adjuntos enviados por el usuario o por sus clientes (imágenes, videos, documentos, etc.)

3.2 Datos técnicos automatizados

  • Dirección IP
  • Registros de servidor (logs)
  • Eventos de automatizaciones (entradas y salidas de los flujos)
  • Marcas de tiempo (timestamps) de ejecuciones
  • Tokens de acceso generados por APIs de terceros

3.3 Accesos autorizados desde APIs de terceros

Según la integración activada, la Aplicación puede acceder a:

  • Instagram / Meta / Facebook: información del perfil comercial, métricas de engagement, contenido de mensajes recibidos (solo si el usuario habilita la automatización del inbox).
  • WhatsApp Cloud API u otras plataformas de mensajería: mensajes entrantes y salientes, medios enviados por el usuario.
  • Servicios de IA y almacenamiento: solicitudes necesarias para el procesamiento de texto, imágenes, archivos y demás operaciones definidas por el usuario.

4. Finalidad del tratamiento

Los datos son procesados únicamente para:

  • Ejecutar las automatizaciones definidas por el usuario.
  • Analizar entradas y generar respuestas mediante modelos de IA.
  • Procesar imágenes, textos o archivos para automatizaciones específicas.
  • Integrar y operar APIs externas solicitadas por el usuario.
  • Mejorar la operación, rendimiento y estabilidad de la Aplicación.
  • Garantizar seguridad, trazabilidad y auditoría de los flujos automáticos.

En ningún caso utilizamos los datos para publicidad no autorizada, venta de información o análisis comercial ajeno a las finalidades aquí descritas.

5. Compartición y transferencia de datos

La Aplicación no vende datos a terceros.

Los datos pueden compartirse únicamente con proveedores necesarios para operar el servicio, por ejemplo:

  • Servicios de hosting y servidores.
  • Proveedores de IA (por ejemplo, plataformas de modelos de lenguaje o visión).
  • Plataformas de mensajería (Meta, WhatsApp, Telegram, etc.).
  • Servicios de almacenamiento de archivos o imágenes.

Asimismo, los datos podrán compartirse cuando exista una obligación legal, para prevenir fraude, abuso o usos indebidos, o para brindar soporte técnico sobre incidentes relacionados con la Aplicación.

6. Tiempo de retención

Los datos se almacenan únicamente durante el tiempo necesario para cumplir con las finalidades del servicio, por ejemplo:

  • Datos de logs y eventos técnicos: entre 30 y 180 días, según la configuración.
  • Mensajes procesados: según las preferencias o flujos definidos por el usuario.
  • Tokens de acceso: hasta que el usuario los revoque o elimine la integración.
  • Archivos y medios: según la automatización o las políticas configuradas por el usuario.

El usuario puede solicitar la eliminación de datos conforme a la sección de derechos del usuario.

7. Seguridad

Implementamos medidas razonables de seguridad para proteger la información, tales como:

  • Uso de HTTPS para el cifrado de datos en tránsito.
  • Almacenamiento seguro de claves API y tokens de acceso.
  • Servidores y contenedores aislados para la ejecución de automatizaciones.
  • Controles de acceso internos y auditoría de actividades relevantes.

Sin embargo, ningún sistema es completamente infalible. Aunque trabajamos para proteger la información, no podemos garantizar seguridad absoluta frente a todas las amenazas.

8. Derechos del usuario

El usuario tiene derecho a:

  • Solicitar acceso a los datos personales que se estén procesando.
  • Solicitar corrección o actualización de sus datos.
  • Solicitar la eliminación total o parcial de sus datos, cuando sea aplicable.
  • Revocar en cualquier momento los accesos concedidos a APIs de terceros.
  • Desactivar o eliminar su cuenta en la Aplicación.

Para ejercer estos derechos, el usuario puede escribir a: jepiogo@hotmail.com.

9. Servicios de terceros

La Aplicación puede integrarse con servicios externos como, entre otros:

  • Meta / Instagram / Facebook
  • WhatsApp Cloud API u otras plataformas de mensajería
  • Plataformas de IA y procesamiento de lenguaje/imágenes
  • Servicios de almacenamiento de archivos o imágenes

Cada uno de estos servicios cuenta con su propia política de privacidad, la cual el usuario acepta al conectar sus cuentas o activar las integraciones en la Aplicación.

10. Cambios a esta Política de Privacidad

Podemos actualizar esta Política de Privacidad para reflejar cambios en la Aplicación, requisitos legales o mejoras en la forma en que gestionamos la información. La versión más reciente estará siempre disponible en esta página.

Recomendamos revisar periódicamente esta Política para mantenerse informado sobre cómo protegemos y tratamos los datos.

Si tienes dudas o comentarios sobre esta Política de Privacidad, puedes contactarnos en: jepilogo@hotmail.com.
